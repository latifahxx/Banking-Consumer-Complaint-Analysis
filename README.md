# Banking Consumer Complaint Analysis

## Project Overview
Analyzed consumer complaints by product, issue type, and resolution timeliness to identify recurring problems and improve service quality.

## Tools Used
- Excel (Pivot Tables, Charts)
- Power BI (Dashboards, KPIs, Interactive Reporting)

## Features
- Tracks complaints by product, issue type, and resolution timeliness
- Identifies recurring problems
- Interactive dashboards for monitoring complaint trends

## Impact
- Highlighted areas for operational improvement
- Supported decisions to enhance customer service
- Provided actionable insights to reduce complaint resolution time

## How to Use
1. Load complaints dataset from `/Data`
2. Explore dashboards in Power BI (`/Reports`)
